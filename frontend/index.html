Export cookies from your TikTok browser session in Netscape format using a browser extension like "Get cookies.txt". Provide the path here to help with geo-restricted or private content.

Troubleshooting

Downloads not working? TikTok changes its API frequently. Update yt-dlp:

pip install -U yt-dlp

Rate limited / 403 errors? Wait a few minutes or add a cookies.txt file above.

Hashtag scraping failing? TikTok hashtag scraping is currently unreliable in yt-dlp. Use username-based bulk fetch instead.
